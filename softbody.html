GPU Accelerated Soft-Body

In my "Advanced Game Tech" module one of the advanced tasks was to create a GPU accelerated soft-body. I created this using a set of points connected to each other by springs. Both the spring interactions and the other physics acting on the particles (colissions with other objects and gravity) are calculated on the GPU. For this project i used C++ and the NCLGL framework from the graphics course.

Each update the soft body is calculated in the following way:

  • Calculate the collisions of each node with other objects in parallel
  • Calculate the spring constraints in 8 separate parralelisable sets. For a set to be parrallel each node can only be affected by one spring
  • Repeat this for 5 passes to achieve a better result
